# Open Multi Links Button
Dieses Skript ist nahezu identisch mit "Open Multi Links". Hierbei gibt es aber zum Öffnen mehrerer Links gleichzeitig statt des 
Kontextmenüeintrages einen Button in der Navbar.

Das **Ergebnis des Skripts**:

![Screenshot Open Multi Links Button](https://github.com/ardiman/userChrome.js/raw/master/openmultilinksbutton/scr_openmultilinksbutton.png)

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
