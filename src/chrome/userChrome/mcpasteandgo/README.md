# MC Paste And Go
Ein Skript, mit dem man den Inhalt der Zwischenablage mit einem Mittelklick oder Shift + Linksklick in eine Suchleiste übergeben kann und die Suche 
startet automatisch.

Das funktioniert in allen Firefoxleisten (Url -, Search-, Findbar) und in den Suchformularen, die viele Seiten selbst zur Verfügung stellen.

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
