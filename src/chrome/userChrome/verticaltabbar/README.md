# Vertical Tabbar
Hiermit werden die Tabs vertikal auf der linken Browserseite angezeigt, Alternative zum Skript  "Vertical Tab".

Um die Tabs untereinander verschieben zu können, muss das Skript "Remove Tab Move Animation" ebenfalls installiert sein.

Das **Ergebnis des Skripts**:

![Screenshot Vertical Tabbar](https://github.com/ardiman/userChrome.js/raw/master/verticaltabbar/scr_verticaltabbar.png)

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
