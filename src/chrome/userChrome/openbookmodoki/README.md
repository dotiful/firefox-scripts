# Open Book Modoki
Vergleichbar mit "Edit Bookmark Plus" können hiermit die Lesezeichen-Panels in ihrer Größe verändert werden. Hinzu kommt aber ein informativ noch 
mehr ausgedehntes Panel zum Vorschein.

Das **Ergebnis des Skripts**:

![Screenshot Open Book Modoki](https://github.com/ardiman/userChrome.js/raw/master/openbookmodoki/scr_openbookmodoki.png)

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
