# Context Style Switcher
Basiert auf der gleichnamigen Erweiterung. Dieses Skript bringt das Men端 "Webseiten-Stil" als Button in die Men端bar (kann aber auch woanders hin) 
und als Men端eintrag ins Hauptkontextmen端.

![Screenshot Context Style Switcher](https://github.com/ardiman/userChrome.js/raw/master/contextstyleswitcher/scr_contextstyleswitcher.png)

Kopiere die uc.xul-Datei in den Chromeordner des Profils.
