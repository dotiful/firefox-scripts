# Restart Firefox Button_Movable
Wie bei den anderen Restart-Skripten ein Neustart des Browsers mit automatischem Leeren des userChrome.js-Cache. Hierbei kann man 
aber die Buttonposition über das "Anpassen"-Fenster frei wählen.

Das **Ergebnis des Skripts**:

![Screenshot Restart Firefox Button_Movable](https://github.com/ardiman/userChrome.js/raw/master/restartfirefoxbutton_movable/scr_restartfirefoxbutton_movable.png)

## Installation
Kopiere die uc.js-Datei in den Chromeordner des Profils.
