# Page Zoom Button
Basiert auf der gleichnamigen Erweiterung, mit dem Unterschied, dass es hier nur einen Button in der Navbar, direkt hinter der Urlbar gibt, der 
alle Zoomfunktionen des Addons bereit stellt.
